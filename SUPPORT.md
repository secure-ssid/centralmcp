# Support

centralmcp is an independent community project. Support is best-effort and
focused on safe, reproducible HPE Networking MCP workflows.

## Before asking for help

1. Check the [documentation site](https://secure-ssid.github.io/centralmcp/).
2. Run the local doctor:

```bash
uv run python scripts/doctor.py
```

3. Check [troubleshooting](docs/troubleshooting.md) for setup, credentials,
   HTTP mode, router catalog, and GitHub Pages deployment notes.

## Where to ask

| Need | Use |
|---|---|
| Reproducible bug | Open a bug report using the GitHub issue form |
| Feature or product workflow idea | Open a feature request using the GitHub issue form |
| Setup, docs, or usage question | Open a GitHub issue with fake or redacted details |
| Security vulnerability or credential exposure | Follow [SECURITY.md](SECURITY.md), not a public issue |
| Pull request guidance | See [CONTRIBUTING.md](CONTRIBUTING.md) |

## No secrets

Do not post real credentials, tokens, tenant IDs, customer data, private URLs,
network diagrams with sensitive details, or screenshots that expose private
information. Use fake hosts, fake IDs, and redacted payloads.
